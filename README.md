# 今日のタスク

その日のタスクを管理する、インストール不要のシンプルなWebアプリＰＷＡです。

## 主な機能

- タスクの追加・完了・削除
- **優先度**（高／中／低）と**時刻**を設定でき、自動で並び替え
- タスクごとの**メモ**欄
- 未完了タスクは翌日へ**持ち越し**
- 完了タスクは日付ごとに**履歴**として記録
- データはブラウザ内（localStorage）に保存
- **PWA**対応（ホーム画面に追加してアプリのように使える／オフライン動作）
- ダークモード対応

## 使い方

`index.html` をブラウザで開くだけで動きます。

ホーム画面アプリとしてインストールする場合は、ローカルサーバー経由（例: `python3 -m http.server`）またはデプロイ先のURLから開き、
- iPhone/Safari: 共有 →「ホーム画面に追加」
- Android/PC Chrome: アドレスバーのインストールアイコン

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（HTML/CSS/JS） |
| `manifest.json` | PWA設定 |
| `sw.js` | Service Worker（オフライン対応） |
| `icon.svg` / `apple-touch-icon-*.png` | アプリアイコン |
